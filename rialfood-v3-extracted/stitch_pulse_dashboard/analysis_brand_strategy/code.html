Brand & Competitive Analysis

1. Competitive Landscape

  • MyFitnessPal: Extensive data but cluttered "legacy" UI.
  • Lifesum: Very inclusive, lifestyle-oriented, excellent habit tracking.
  • Cronometer: Technical "Bio-hacker" favorite; deep micronutrient focus but utilitarian design.
  • Whoop/Oura: The benchmark for performance data visualization and premium, dark-mode aesthetics.

2. Brand Refinement: "VitalityEngine"

To appeal to both athletes and families, we need to balance Performance with Approachability.

  • The Athlete: Needs Whoop-style data mapping food to recovery.
  • The Parent: Needs "Kid-Approved" tags and simple, high-quality recipe visuals.
